# AramChat
아람챗은 마인크래프트에서 사용 가능한 한글 채팅 모드입니다. 한/영 전환 버튼을 사용해 한글 채팅을 할 수 있고 한글 입력 상태가 다른 키를 방해하지 않습니다

## 기능
* 한영 키로 한글/영어 전환 가능
* 한글 상태에서도 모든 키를 조작 가능
* 자판 상태 인디케이터 기능

## 옵션
* 인 게임 키 설정 란에서 한글/영어 전환 키를 변경할 수 있습니다

## 미구현 기능
* 세벌식 키보드 자판
* 한자 키 기능
* 폰트 변경 기능
